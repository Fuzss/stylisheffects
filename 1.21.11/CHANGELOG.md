# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v21.11.3-1.21.11] - 2025-12-30

### Fixed

- Fix mob effects with an amplifier of zero rendering that amplifier behind their name (as `I`)
- Fix mob effect names sometimes being cut off when setting a small widget scale in the config

## [v21.11.2-1.21.11] - 2025-12-29

### Added

- Add a missing option for turning off effect duration text

## [v21.11.1-1.21.11] - 2025-12-29

### Added

- Add effect bars on the widgets; very much inspired by
  the [Status Effect Bars](https://modrinth.com/mod/status-effect-bars) mod

### Changed

- Greatly streamline the config; removing many previously duplicated settings

### Removed

- Remove outdated api modules and events

## [v21.11.0-1.21.11] - 2025-12-28

### Changed

- Update to Minecraft 1.21.11
